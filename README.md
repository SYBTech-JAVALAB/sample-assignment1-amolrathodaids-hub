## Topic
Create a class for an online bookstore inventory system featuring attributes for:
- Title
- Author
- Price
- ISBN
- Current stock level

---

## Objective
The objective of this program is to design and implement a Java class that represents books in an online bookstore and supports inventory and pricing operations.

---

## Features of the Program

### 1. Display Book Information
- Presents all book details (title, author, price, ISBN, stock level)
- Output is formatted clearly and user-friendly

### 2. Apply Promotional Discount
- Reduces the book price by a specified percentage
- Useful during sales or promotional offers

### 3. Update Stock Level
- Increases stock count when new copies arrive
- Helps maintain accurate inventory records

### 4. Check Availability
- Determines whether a book is available for purchase
- Based on current stock level

---

## Implementation Details
- A `Book` class is created with required attributes
- Methods are implemented for:
  - Displaying book details
  - Applying discount
  - Adding stock
  - Checking availability
